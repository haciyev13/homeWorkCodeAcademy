Front-end, web sitelerinin veya uygulamalarının kullanıcı tarafında yer alan görüntü ve etkileşimleri ile kullanıcı deneyimini tasarlayan ve geliştiren bir web geliştirme disiplinidir. Front-end geliştiricileri, web sayfalarının tasarımını, animasyonlarını, kullanıcı arayüzünü ve interaktif özelliklerini kodlayarak, web sitelerinin veya uygulamaların kullanıcılar tarafından kullanılabilir, anlaşılabilir ve etkileşimli hale getirilmesine yardımcı olurlar. Front-end geliştirme, HTML, CSS ve JavaScript gibi web teknolojilerini kullanır ve web sayfalarının uygun görüntülenmesi ve kullanıcı etkileşimleri için tarayıcılar tarafından desteklenen standartlara uyar. Bu disiplin, web tasarımı, kullanıcı deneyimi tasarımı, web programlama ve web yazılım geliştirme gibi diğer web geliştirme alanları ile birlikte çalışır ve bir web sitesinin veya uygulamanın tümünün kullanıcı dostu ve işlevsel olarak çalışmasını sağlar.
